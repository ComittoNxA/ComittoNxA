Image processing number of threads

Initial setting of the image processing number of threads is 2 thread.
To change to the number of cores terminal of CPU, image processing might be faster.
Please set in [Detail Settings for Image Viewer]-[Max number of processing threads].

Immersive Mode

Setting the disabled as follows:[Detail Setting for Image & Text]-[Enable Immersive Mode].

Directory List

The directory is added when you select "Regist Directory" from the menu in file list.
To display by the flick of a list or icon selection at the bottom of the screen.

Bookmark List

The bookmark is added when you select "Add Favorite" from the menu in text or image viewer.
To display by the flick of a list or icon selection at the bottom of the screen.

History List

Displays the information of the files that are open.
To display by the flick of a list or icon selection at the bottom of the screen.

Support scope of PDF

Text is not displayed.
Instead, You can view the PDF self-catering in the NAS.
